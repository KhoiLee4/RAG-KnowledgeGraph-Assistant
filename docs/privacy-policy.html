Chính sách quyền riêng tư

Ứng dụng: RAG Knowledge Graph Assistant (đồ án tốt nghiệp)

Cập nhật: 2026

1. Dữ liệu thu thập

Ứng dụng sử dụng đăng nhập Google OAuth để truy cập Google Drive (chỉ đọc) của bạn nhằm đồng bộ và lập chỉ mục tài liệu phục vụ hỏi đáp AI. Chúng tôi lưu token OAuth và metadata tài liệu trên máy chủ bạn tự triển khai.

2. Mục đích sử dụng

  • Đọc file bạn chọn đồng bộ từ Google Drive
  • Tạo embedding và knowledge base cá nhân để trả lời câu hỏi

3. Chia sẻ dữ liệu

Dữ liệu Drive không được bán hoặc chia sẻ cho bên thứ ba, ngoại trừ Google Gemini API để xử lý embedding và sinh câu trả lời (theo cấu hình API key của bạn).

4. Lưu trữ & xóa

Token và tài liệu đã index lưu trên server local (ChromaDB, Neo4j). Bạn có thể đăng xuất hoặc xóa tài liệu khỏi ứng dụng bất cứ lúc nào.
